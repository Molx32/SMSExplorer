Project description

  • SMS Explorer aims at demonstrating the risk of temporary and public virtual phone numbers by analysing public SMSs and categorizing them (c.f. table below). For more information, take a look at the detailed blog post.

    Is target interesting? Label Description
    ✅ DATA_INTERESTING_DESC_YES_PII This category refers to SMSs that contain Personal Identifiable Information (PII). This information may be visible either in the mesasge or by navigating to a URL. An example of that can be found here.
    ✅ DATA_INTERESTING_DESC_YES_DISCOVERY This category refers to SMSs that contain sensitive information that may allow an attacker to gain information on a target. An example of that can be found here.
    ✅ DATA_INTERESTING_DESC_YES_ACCOUNT_TAKEOVER This category refers to SMSs that may lead to account takeover. An example of that can be found here.
    ✅ DATA_INTERESTING_DESC_YES_ACCOUNT_PASSWORD_RESET This category refers to SMSs that may lead to account takeover, using password reset URLs only. An example of that can be found here.
    ❌ DATA_INTERESTING_DESC_NO_SCAM This category refers to SMSs that try to impersonate legitimate SMS from official enterprises, or SMS that contains promotions for gambling for instance. An example of that can be found here.
    ❌ DATA_INTERESTING_DESC_NO_AD This category refers to SMSs that simply contain ads. An example of that can be found here.
    ❌ DATA_INTERESTING_DESC_NO_MISSED_OPPORTUNITY This category refers to SMSs that would be exploitable if the message wasn't cropped. An example of that can be found here.
    ❌ DATA_INTERESTING_DESC_NO_OTHER This category refers to other SMSs that are not interesting e.g. a link to download an application.
